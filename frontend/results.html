Результаты подбора стипендий

Вы уже можете претендовать на:

    {% if scholarships.eligible %} {% for scholarship in scholarships.eligible %}
  • {{ scholarship }}
    • {% endfor %} {% else %}
  • К сожалению, подходящих стипендий не найдено
    • {% endif %}

Вы можете претендовать, если выполните дополнительные условия:

    {% if scholarships.potential %} {% for scholarship in scholarships.potential %}
  • {{ scholarship }}
    • {% endfor %} {% else %}
  • Нет дополнительных стипендий для отображения
    • {% endif %}
